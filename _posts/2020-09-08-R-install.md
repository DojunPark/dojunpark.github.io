## 1. R 공식 홈페이지에 접속하여 알맞은 버전으로 다운로드 
① https://www.r-project.org/ 접속
② Download R, 또는 좌측 메뉴의 CRAN 클릭
③ Korea 목록에 있는 링크로 접속
④ 운영체제에 맞는 R installer 다운로드 (e.g. R for Windows, Linux)

## 2. R installer 실행
① 설치 언어 선택 (e.g. 한국어, English)
② 설치할 위치 선택
③ 구성 요소 선택 (64bit 컴퓨터의 경우 32bit 설치 제외)
④ 스타트업 옵션 기본값 설정
⑤ 추가 사항 적용 (e.g. 바탕화면에 아이콘 생성)

## 3. R Studio 설치
① R Studio 홈페이지(https://rstudio.com/)에서 하단의 Product – open source - R Studio Desktop 클릭
② open source edition(무료 버전) 클릭
③ Download RStudio Desktop for Windows 클릭하여 설치 프로그램 다운로드

## 4. RStudio installer 실행
설치할 파일 위치 선택 후 프로그램 설치 진행
