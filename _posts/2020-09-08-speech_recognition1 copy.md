## 1. 데이터 구조의 이해
알라딘, 자스민, 지니라는 세 사람이 있다고 가정해보자. 알라딘은 남자이고, 20세, 그리고 70, 80이라는 두 값을 가지고, 자스민은 여자, 19세, 85, 90라는 두 값을, 지니는 남자, 30세, 90, 75라는 두 숫자를 가진다. <br/>


<a  href="https://imgur.com/DWlJ5By"><img  src="https://i.imgur.com/DWlJ5By.png"  width="500px"  title="source: imgur.com"  /></a><br/>


이를 엑셀로 정리하여 정리하자면 다음과 같이 표현할 수 있다. 이러한 경우 각각의 항목을 나타내는 이름, 성별, 나이와 같은 라벨을 붙여줄 수 있다. <br/>
R의 자료는 크게 배열, 벡터, 리스트, 요인, 데이터프레임과 같이 6개로 구분할 수 있다. 위에서 예시로 보여준 데이터의 경우 테이블 형태로 저장이 되는 데이터 프레임의 형태에 해당된다. <br/>
고전적인 방법에서는 각각의 항목의 값이 벡터로 정렬되어 저장되고, 같은 단위(동일 차원)라는 조건 하에 여러 벡터가 모이면 배열이 되고, 여러 배열이 모여 3차원 배열이 구성된다. <br/>
숫자 정보 이외에 성별과 같이, 구분할 수 있는 값을 요인(factor)라고 부른다. <br/>

## 2. 벡터와 연산자
알라딘의 정보가 나열된 하나의 행을 “개체”라고 부르며, 하나의 개체가 가진 각각의 항목을 “변수”라고 부른다. 따라서 위 데이터프레임은 3 개체, 5 변수로 구성되어 있다고 설명할 수 있다. <br/>
위 예시의 행은 각 단위가 다르기 때문에 벡터로 인식하지 못하며, 열은 동일한 단위로 구성되기에 벡터로 볼 수 있다. <br/>
하나의 벡터를 X라는 이름으로 부를 때, 첫번째 값은 X[1], 두번째 값은 X[2], 세번째 값은 X[3]으로 명칭하여 불러올 수 있다. <br/>
‘’’R
setwd("c:\\data")    # C드라이브에 data 폴더를 working directory로 설정
score <- 70           # score 변수에 70이라는 값을 대입
score <- c(70, 85, 90)  # score 변수에 벡터를 대입
score[3] <- 95        # score 변수의 세번째 값을 95로 변경
score[4] <- 100       # score 변수에 네번째 값을 추가
name <- c("알라딘", "자스민", "지니")  # 새로운 변수에 벡터를 추가
‘’’

식별자(변수명)는 변수 또는 함수 등을 다른 것들과 구별하기 위해 사용되는 ‘이름’을 지칭하는 용어이다. 식별자의 이름은 문자와 숫자, ‘.’, ‘_’으로 구성할 수 있지만, 숫자나 ‘_’로 시작할 수 없다. 또한 R에서 정의되어 있는 예약어 역시 식별자로 사용할 수 없다. <br/>

## 실습 1 - 연속적인 값들의 벡터 만들기
'''R
x <- seq(1, 10, by=3)   # 1부터 10까지 3간격으로 입력
x <- 1:10              # 1에서부터 10까지 입력
x <- 10:1              # 10에서 1까지 역순으로 입력
x <- seq(1, 10, length.out=5)   # 1부터 10까지 동일한 간격으로 5개의 값을 입력
'''

## 실습 2 - 반복적인 값들의 벡터 만들기
'''R
x <- c(1, 2, 3)           # x에 벡터를 입력
y <- rep(x, times = 2)   # y에 x 벡터를 2번 입력 (1, 2, 3, 1, 2, 3)
y <- rep(x, each = 2)    # y에 x 벡터를 각각 2번 입력 (1, 1, 2, 2, 3, 3)
'''

## 실습 3 - 연산자
'''R
3 + 2   # 더하기 연산자
3 – 2   # 빼기 연산자
3 * 2   # 곱하기 연산자
3 / 2   # 나누기 연산자
3 ** 2   # 거듭제곱 연산자1
3^2     # 거듭제곱 연산자2
3 %% 2   # 나머지 연산자
3 %/% 2   # 몫 연산자
3 > 2     # 비교연산자, 큼 (TRUE)
3 >= 2   # 비교연산자, 크거나 같다 (TRUE)
3 < 2    # 비교연산자, 작다 (FALSE)
3 <= 2   # 비교연산자, 작거나 같다 (FALSE)
3 == 2   # 비교연산자, 같다 (FALSE)
3 != 2   # 비교연산자, 같지 않다 (TRUE)
TRUE | FALSE   # 논리합, 두 값 중 하나가 TRUE면 TRUE (TRUE)
TRUE & FALSE  # 논리곱, 두 값 모두가 TRUE여야 TRUE (FALSE)
!TRUE          # 논리부정 (FALSE)
isTRUE(TRUE)   # 진위여부 (TRUE)
'''

## 실습 4 - 벡터와 사칙연산
'''R
x <- c(10, 20, 30, 40)    # 변수 x에 4자리 벡터를 입력
y <- c(2, 4, 6, 8)        # 변수 y에 4자리 벡터를 입력
z <- c(2, 4)            # 변수 z에 2자리 벡터를 입력
x + 5     # 벡터의 각 값에 5를 더함, [1] 15 25 35 45
x + y     # 두 벡터를 같은 자리끼리 더함, [1] 12 24 36 48, 
x + z     # 자동으로 벡터의 차원을 확장하여 더함 (broadcasting), [1] 12 24 32 44
'''

## 실습 5 - 비교연산자
'''R
x <- c(10, 20, 30)   # 변수 x에 벡터를 입력
x <= 10           # x 벡터의 각 값이 10보다 작거나 같은지, [1]  TRUE FALSE FALSE
x[x>15]           # x 벡터의 값 중 15보다 큰 값을 출력, [1] 20 30
any(x <= 10)      # x 벡터의 값 중 10보다 작거나 같은 값이 있는지, TRUE
all(x <= 10)       # x 벡터의 모든 값이 10보다 작거나 같은 지, FALSE
xor(x, y)          # 논리연산자, 두 값이 같으면 FALSE, 다르면 TRUE
'''

## 실습 6 – 결측치와 널
'''R
x <- NULL   # x에 NULL을 입력
is.null(x)     # x가 NULL인지 진위여부 판단, TRUE
y <- c(1, 2, 3, NA, 5)   # y에 결측치가 포함된 벡터를 입력
z <- 10/0    # 10을 0으로 나눔, [1] Inf (불능)
u <- 0/0     # 0을 0으로 나눔, [1] NaN (not a number의 약자, 부정)
'''
-	결측치 - NA(not available), 데이터가 누락된 상태
-	NULL - 변수명만 있고, 값이 아직 할당되지 않은 상태
